UserConfig Methods

  • – resetFormats

    Remove all patterns and apply clear format style. Default format is @“#############”, imagePath is nil.
  • – setDefaultFormat

    Remove all patterns and apply default format style. Default format is “## (###) ###–##–##”, imagePath is nil.
  • – setPredefinedFormats

    Remove all patterns and set predefined values. Method contains values for next countries: Russia, Kazakhstan, Belorussia, Ukraine, Armenia, Tajikistan, Kirgistan, Uzbekistan, Moldova, Azerbaijan, Turkmenistan Default format is “#############”, imagePath is nil.
  • – setDefaultOutputPattern:imagePath:

    Apply default format style and image Symbol ‘#’ assumes all digits and ‘+’. Example is “## (###) ###–##–##”, imagePath is “flag_ru”.
  • – addOutputPattern:forRegExp:imagePath:

    All number matched your regexp will formatted with your style and image Symbol ‘#’ assumes all digits and ‘+’. Example: pattern is “## (###) ###–##–##”, imagePath is “flag_ru”, regexp is “^\+375\d*$”
