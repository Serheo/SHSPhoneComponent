Overview

Simple UITextField subclass to handle phone numbers formats ARC enabled.

Tasks

  • – resetFormats

    Remove all patterns and apply clear format style. Default format is @“#############”, imagePath is nil.
  • – setDefaultOutputPattern:imagePath:

    Apply default format style and image Symbol ‘#’ assumes all digits. Example is “+# (###) ###–##–##”, imagePath is “flag_ru”.
  • – addOutputPattern:forRegExp:imagePath:

    All number matched your regexp will formatted with your style and image Symbol ‘#’ assumes all digits. Example: pattern is “+# (###) ###–##–##”, imagePath is “flag_ru”, regexp is “^\+375\d*$”
  • – setLogicDelegate:

    If you want to use delegate methods you need to subclass SHSPhoneLogicDelegate and call setLogicDelegate: method. It initialize your delegate subclass with required parameters. Be careful with textField:shouldChangeCharactersInRange:replacementString: – it should always return NO.
  • – setTextDidChangeBlock:

    Block will be called when text changed
